## ⚡ Installation (PC / Mac / Linux)

1. `01-INSTALL-DoubleClick.py` — Doppelklick (Python 3.8+ erforderlich)
2. `LLR-Revival-5.0-XX.html` — im Browser öffnen

**Tablet/Smartphone:** Fertigen Ordner vom PC kopieren — kein Python, keine Installation!

## 📋 Voraussetzungen
